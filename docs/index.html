기초 파이썬 개발 환경 구성¶

  • 윈도우즈 (10 & 11)에서는 마이크로소프트 스토어를 통해 Python 3.10을 설치합니다.

  • 리눅스 우분투나 데비안에서는 sudo apt install -y python3를 사용해서 설치합니다.

이후 파이썬 가상환경을 구성합니다:

python3 -m venv .pyenv

우분투의 경우 sudo apt install -y python3-venv로 venv 설치를 먼저 완료해야 합니다.
