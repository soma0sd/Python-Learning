# 파이썬 학습자료 소스코드 저장소

작성자: soma0sd@gmail.com

고등학교 방과후 프로그래밍 수업과 대학교 전산물리학 조교 때 만든 예제, 전산물리교재연구 수업 때 만든 예제들을 모은 저장소입니다. 저장소의 내용은 [Sphinx를 이용한 문서화](https://soma0sd.github.io/Python-Learning)를 진행하는 중입니다.
